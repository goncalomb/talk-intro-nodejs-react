### What is Node.js

<Appear>

JavaScript outside the Browser.

* V8 Engine (Google)
* Event Loop
* APIs (stream, os, fs, net, http, crypto, ...)

Asynchronous event-driven runtime with non-blocking I/O.

</Appear>

### Install Node.js (LTS 12.15.0)

<CenterContainer>

https://nodejs.org/en/<br/>
https://nodejs.org/en/download/package-manager/<br/>

**MacOS**<br/>
Download the Installer or using Brew:

```bash
brew install node
```

**Linux Debian** <small>(NodeSource Binaries)</small>:

```bash
curl -sL https://rpm.nodesource.com/setup_12.x | sudo bash -
```

**Windows**<br/>
Download the Installer or using Chocolatey:

```bash
choco install nodejs.install
```

</CenterContainer>
